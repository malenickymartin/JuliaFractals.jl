SPJFractals.create_newton_equation — Method
create_newton_equation(func::Function)

Create a Newton fractal equation from a differentiable function f(x), solving f(x) = 0. Each root is assigned a unique color based on the angle of the complex number.

Arguments

  • func::Function: The function to create the Newton fractal equation from.

Returns

A function with input parameters x, y, and params, that computes the Newton fractal.
